# Design Workspace Guide

## Current Status

The repository is undergoing a full redesign under feature `004-design-language-store`.

- The active planning and implementation rules for the redesign live in `specs/004-design-language-store/`.
- Files in this folder may be either current design-definition artifacts or legacy references.
- Legacy references must be explicitly marked so they are not mistaken for the active source of truth.

## Artifact Roles

- `DESIGN.md`: legacy reference for the superseded "Rhythmic Rail" direction
- `code.html`: reference mock from an earlier design pass unless updated and explicitly re-designated
- `screen.png`: snapshot of the earlier reference mock

## Working Rule

When the current redesign defines new design-system artifacts, they should be added here with clear ownership and status so collaborators can tell which files are active and which are historical.
